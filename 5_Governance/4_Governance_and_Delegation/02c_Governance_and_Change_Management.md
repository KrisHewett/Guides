## Change Management Process

### 1. Change Request Initiated
- A change request is submitted.
- [Clinical Governance INBOX](mailto:Health.DHWClinicalGovernanceEnquiries@sa.gov.au)
  - **Responsible Position:** Requesting Stakeholder/Team

### 2. Identify Target Zone
- Determine if the change affects the CAE13 Collaboration Zone.
  - **Responsible Position:** Requesting Stakeholder/Team

### 3. Route to CAE13 Developer/Admin
- If CAE13 is affected, forward the request to the CAE13 Developer/Admin.
  - **Responsible Position:** DAP Central Support Services

### 4. Review and Scope Validation
- Assess the request for accuracy, relevance, and alignment with current work.
  - **Responsible Position:** CAE13 Developer/Admin

### 5. Technical Design and Planning
- Develop a technical plan including dependencies, risks, and implementation steps.
  - **Responsible Position:** CAE13 Developer/Admin

### 6. Validate Request
- Confirm whether the request is valid and scoped correctly.
  - **Responsible Position:** CAE13 Developer/Admin

### 7. Decision Point
- If clarification is needed, proceed to Step 8a.
- If rejected, proceed to Step 8b.
- If approved, proceed to Step 9.
  - **Responsible Position:** CAE13 Developer/Admin

### 8.a. Request Clarification
- Loop back to Step 4 for re-review after clarification.
  - **Responsible Position:** CAE13 Developer/Admin

### 8.b. Reject Request
- Escalate if necessary.
  - **Responsible Position:** CAE13 Developer/Admin

### 9. Implement Change
- Execute the approved change in the CAE13 environment.
  - **Responsible Position:** CAE13 Developer/Admin

### 10. Notify Stakeholders
- Communicate the change via the designated Teams channel (e.g., SLS Notifications).
  - **Responsible Position:** CAE13 Developer/Admin
    - [General | SAH-DAP-CAE13-Notifications | Microsoft Teams](https://teams.microsoft.com/l/team/19%3AkBs4hF1DKkFlb2I5xfols8psWqDLXm4hdmZG-tjosrw1%40thread.tacv2/conversations?groupId=95844b65-d532-4b41-a0bd-3bfad0d7149d&tenantId=bda528f7-fca9-432f-bc98-bd7e90d40906)
    - [NSQHS Notifications | SAH-DAP-CAE13-Notifications | Microsoft Teams](https://teams.microsoft.com/l/channel/19%3A224c615edca44644baad476a99374e86%40thread.tacv2/NSQHS%20Notifications?groupId=95844b65-d532-4b41-a0bd-3bfad0d7149d&tenantId=bda528f7-fca9-432f-bc98-bd7e90d40906)
    - [SLS Notifications | SAH-DAP-CAE13-Notifications | Microsoft Teams](https://teams.microsoft.com/l/channel/19%3Ad6ab4c3073d94c78a654a1887d0307c4%40thread.tacv2/SLS%20Notifications?groupId=95844b65-d532-4b41-a0bd-3bfad0d7149d&tenantId=bda528f7-fca9-432f-bc98-bd7e90d40906)

### 11. Document and Close
- Finalise documentation and notify stakeholders of completion.
  - **Responsible Position:** CAE13 Developer/Admin
