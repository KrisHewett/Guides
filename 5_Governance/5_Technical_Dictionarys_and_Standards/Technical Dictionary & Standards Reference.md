## **Purpose**

Provide a harmonised, standardised reference set of programming terminology, technical dictionaries, domain‑specific glossaries, and internationally recognised data/metadata standards to support analytics, governance, software development, workflow documentation, and audit readiness.

## **1. Programming Foundations**

### **1.1 Core Components**

- **Subroutines** — Reusable procedural code units.
- **Macro definitions** — Compile‑time expansion mechanisms.
- **Global variables** — Variables with application‑wide scope.
- **Class definitions** — Object‑oriented blueprints defining behaviours & properties.
- **Templates** — Generic programming constructs for type abstraction.

### **1.2 Common Functionality**

- **Algorithms** — Sorting, searching, optimisation.
- **Data structures** — Lists, trees, queues, stacks, hash tables.
- **External interaction** — Input/output, data exchange protocols.
- **Operating system integration** — Processes, environment variables, filesystem access.
